# 需求文档

## 1. 应用概述

- **应用名称**：萌宠缘分测试
- **应用描述**：一款基于生日星座与性格测试的萌宠推荐 H5 应用。用户依次完成生日填写、性格问答后，系统根据结果推荐匹配的萌宠，用户可通过左滑（不心动）/ 右滑（心动）对推荐萌宠进行互动筛选。

---

## 2. 用户与使用场景

- **目标用户**：对宠物感兴趣、希望通过趣味测试找到适合自己萌宠的用户
- **核心使用场景**：用户打开 H5 页面，依次填写生日、完成性格问答，最终获得个性化萌宠推荐列表，并通过滑动交互表达心动或不心动

---

## 3. 页面结构与核心功能

### 3.1 页面总览

```
萌宠缘分测试
├── 第一页：生日收集页
├── 第二页：性格问答页
└── 第三页：萌宠推荐页
```

### 3.2 第一页：生日收集页

**功能说明：**

- 页面顶部展示应用标题与简短引导文案（如「测测你的专属萌宠」）
- 提供生日输入控件，用户选择或填写出生年月日
- 系统根据生日自动计算对应星座，无需用户手动选择
- 底部设置「开始测试」按钮，点击后跳转至第二页
- 校验规则：生日为必填项，未填写时点击按钮提示「请先填写您的生日」

### 3.3 第二页：性格问答页

**功能说明：**

- 展示一组性格测试题目，题目数量建议 5～8 题
- 每题包含题干与选项（单选），用户逐题作答
- 页面顶部展示答题进度（如「第 3 题 / 共 6 题」）
- 支持返回上一题修改答案
- 所有题目作答完成后，展示「查看我的萌宠」按钮，点击跳转至第三页
- 校验规则：每题必须选择后方可进入下一题

**题目示例（可按实际调整）：**

| 题号 | 题干 | 选项 |
|------|------|------|
| 1 | 你的日常生活节奏是？ | A. 规律作息，喜欢稳定 / B. 随性自由，喜欢变化 |
| 2 | 你更喜欢哪种相处方式？ | A. 黏人陪伴型 / B. 独立自在型 |
| 3 | 你的居住环境是？ | A. 较小的公寓 / B. 宽敞的房子或有院子 |
| 4 | 你能接受的互动频率是？ | A. 随时互动，越多越好 / B. 偶尔互动，各自安静 |
| 5 | 你对宠物叫声的接受度？ | A. 完全没问题 / B. 希望安静一些 |

### 3.4 第三页：萌宠推荐页

**功能说明：**

- 系统根据星座 + 性格问答结果，推荐 3～5 只匹配萌宠
- 每只萌宠以卡片形式展示，内容包括：
  - 萌宠名称（如「英短蓝猫」）
  - 萌宠图片
  - 简短匹配说明（如「你们都喜欢安静，超级合拍！」）
  - 匹配度标签（如「星座契合度 ★★★★☆」）
- **滑动交互：**
  - 右滑：心动，卡片向右飞出，显示「心动」标记
  - 左滑：不心动，卡片向左飞出，显示「划走」标记
  - 支持点击页面底部「心动」/「不心动」按钮触发同等效果
- 所有卡片滑动完毕后，展示汇总结果：
  - 列出所有「心动」萌宠名称
  - 展示「你最有缘分的萌宠是 XXX」结论文案
  - 提供「重新测试」按钮，点击返回第一页

### 4.2 萌宠推荐匹配逻辑

- 系统综合星座属性（如火象/土象/风象/水象）与性格问答得分，映射至对应萌宠类型
- 推荐萌宠池示例（可扩展）：

| 萌宠 | 适合特征 |
|------|----------|
| 英短蓝猫 | 喜欢安静、独立、小空间居住 |
| 柯基犬 | 活泼好动、喜欢互动、有活动空间 |
| 仓鼠 | 生活节奏规律、空间有限、喜欢观察 |
| 兔子 | 温和安静、喜欢陪伴但不过度依赖 |
| 鹦鹉 | 喜欢互动、接受叫声、有耐心 |

- 每次推荐结果从匹配萌宠池中筛选 3～5 只，按匹配度排序展示

### 4.3 页面流转规则

- 流程为单向线性：第一页 → 第二页 → 第三页
- 第三页「重新测试」按钮返回第一页，清空所有已填写数据
- 不支持跨页跳转

---

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 生日未填写即点击「开始测试」 | 提示「请先填写您的生日」，阻止跳转 |
| 题目未选择即点击「下一题」 | 提示「请选择一个选项后继续」，阻止跳转 |
| 所有萌宠均左滑（无心动） | 汇总页显示「暂时没有心动的萌宠，要不要重新测试？」并提供重新测试入口 |
| 所有萌宠均右滑（全心动） | 汇总页展示全部心动萌宠，并标注「你是萌宠全能爱好者！」 |

---

## 6. 验收标准

- 生日输入后，系统正确计算并记录对应星座
- 性格问答页进度条随答题推进实时更新
- 每题必须选择后方可进入下一题，未选择时有明确提示
- 萌宠推荐页卡片支持左滑（不心动）与右滑（心动）交互，动效流畅
- 点击底部「心动」/「不心动」按钮与滑动效果一致
- 所有卡片处理完毕后，正确展示汇总结果页
- 「重新测试」按钮点击后返回第一页，所有数据清空
- 整体页面在主流移动端浏览器中正常展示与交互
